# Social Media Marketing Platform

A responsive, multi-page business website built from scratch to showcase clean HTML architecture and custom structural layout engineering.

## 🚀 Live Demo
👉 [Click here to view the live website] https://hussainranadev-aim.github.io/Social_Media_Marketing_Website-Practice/

---

## 🛠️ Tech Stack & Concepts Applied
* **HTML5:** Semantic structure, multi-page cross-navigation, forms, and media asset tracking.
* **CSS3:** Custom layout engineering, typography, structural styling, and interactive hover effects.
* **Layout Architecture:** Built using a custom structural grid system with strict element-isolation principles.
* **Responsive Design:** Utilizes mobile-first media query breakpoints (`@media(max-width: 768px)`) to collapse desktop rows into safe vertical stacks.

---

## 📦 Features Included
* **Multi-Page Architecture:** 4 separate, fully linked pages maintaining a unified brand theme via a single external stylesheet.
* **Isolated Component Design:** Keeps background layout logic strictly separated from content styling classes for cleaner code maintenance.
* **Responsive Layout:** Automatically adjusts navigation elements and card sections smoothly across desktop and mobile screens.

---

## 📈 Learning Milestones
This project served as a major step in my development journey, helping me master:
1. Managing complex asset paths across multiple HTML files.
2. Resolving browser rendering bugs caused by mixing layout boundaries with component padding.
3. Writing clean, maintainable `@media` blocks for mobile layouts.
